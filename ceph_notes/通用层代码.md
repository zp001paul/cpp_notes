## object_t

struct object_t; 代表bluestore层一个对象
struct sobject_t; 在object_t之上多了snapshot信息
struct hobject_t; 在sobject_t之上又多了哈希信息

```c++
struct sobject_t {
    snapid_t snap; // 如果一个对象不是快照对象，就是head对象，这个字段就是CEPH_NOSNAP
}
```
